# 诗词社区 - 沉浸式 UI 改版实施指南

## 📋 改版概述

本次改版以"竹"为核心视觉元素，融合中华传统美学，实现类似"小黑盒"的沉浸式布局效果。

### 核心设计理念
- **统一背景层**：竹影水墨渐变，贯穿全站
- **毛玻璃导航栏**：半透明悬浮，glassmorphism 效果
- **融合式内容**：去除硬边框，内容自然融入背景
- **Hero 区整合**：将"每日一首"作为首屏的一部分

---

## 🎨 设计 Token 系统

### 色彩系统
```scss
// 主色调：竹青系
$color-bamboo-primary: #5a8c6f;      // 竹青
$color-bamboo-light: #7ba88a;        // 浅竹青
$color-bamboo-dark: #3d5a4a;         // 深竹青

// 辅助色：墨色系
$color-ink-black: #2c3e50;           // 墨黑
$color-ink-gray: #5a6c7d;            // 墨灰
$color-ink-light: #95a5a6;           // 淡墨

// 背景色：宣纸系
$color-paper-white: #faf9f6;         // 宣纸白
$color-paper-beige: #f5f3ed;         // 米白
$color-paper-cream: #ebe8df;         // 奶白

// 点缀色
$color-accent-gold: #d4af37;         // 印章金
$color-accent-red: #c8553d;          // 朱砂红
```

### 字体系统
- **中文字体**: PingFang SC, Microsoft YaHei, Noto Sans SC
- **诗词专用**: STKaiti, KaiTi, FangSong (衬线字体)
- **字号基准**: 8px 步进 (12px ~ 40px)
- **行高**: 1.4 (紧凑) ~ 2.0 (宽松)

### 间距系统
- **4px 步进**: 4px, 8px, 16px, 24px, 32px, 48px, 64px
- **圆角**: 4px ~ 24px
- **阴影**: 轻量 → 中等 → 重度

## 🎨 设计规范

### 布局规则
1. **间距统一**: 所有模块间距使用 4px 步进
2. **最大宽度**: 内容区最大 1200px，容器最大 1400px
3. **卡片圆角**: 统一使用 12px ~ 24px
4. **阴影层级**: 轻量 (卡片) → 中等 (悬浮) → 重度 (模态框)

### 颜色使用
1. **主色调**: 竹青色 (#5a8c6f) 用于按钮、链接、强调
2. **文字色**: 墨黑 (#2c3e50) 标题，墨灰 (#5a6c7d) 正文
3. **背景色**: 宣纸白 (#faf9f6) 主背景，半透明白卡片
4. **点缀色**: 印章金 (#d4af37) 特殊按钮，朱砂红 (#c8553d) 警告

### 动效原则
1. **过渡时间**: 150ms (快速) ~ 350ms (慢速)
2. **缓动函数**: cubic-bezier(0.4, 0, 0.2, 1)
3. **Hover 效果**: 轻微上浮 (-2px ~ -4px) + 阴影增强
4. **避免过度**: 不超过 300ms，保持温和文雅
